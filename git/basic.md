### 최초 설정

처음 컴퓨터에 git 설치하면, 사용자의 이메일과 이름을 적어준다. 이는 앞으로 일어나는 커밋에 서명을 하기 위해서 필요하다.

```
$ git config --glober user.name "<hongsinkang>"

$ git config --glober user.email "<hongsinkang>@<gmail.com>"
```

잘 설정되었나 확인하려면
```
$ git config user.name
이름 출력

$ git config user.email
이메일 출력
```
